# Frontend Development Guidelines

> Best practices for frontend development in the Zhuīyì (追忆) project.

---

## Overview

This project is a Next.js 14 single-page application with four states (LANDING → PROCESSING → EXPERIENCE → SHARE). The frontend uses React, Tailwind CSS, Framer Motion, Zustand, and AMap (Amap/Gaode Maps).

## Pre-Development Checklist

Before writing any frontend code, read:

1. **Directory Structure** — know where to put files
2. **Component Guidelines** — know the component patterns
3. **State Management** — know where state lives
4. **Quality Guidelines** — know what's forbidden
